# Release v2.1.0-beta.1

Це **beta**-реліз для перевірки релізного процесу й документації перед наступним стабільним релізом.

## Що входить
- Окрема релізна гілка: `release/v2.1.0-beta.1`.
- Оновлення `README.md`: уточнено, що синхронізація з Google Sheets запускається вручну з адмін-панелі.

## Як тестувати
- Запуск: `python main.py`.
- Перевірити базові сценарії бота (старт/стоп зміни, паливо, ТО).
- Перевірити ручну синхронізацію через адмін-панель.

## Відомі обмеження
- Це beta: можливі зміни без зворотної сумісності до стабільного релізу.
